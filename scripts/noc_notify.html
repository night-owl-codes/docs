noc_notify

A beautiful, customizable notification system with a western aesthetic. Perfect for RedM servers that want period-appropriate UI elements.

The noc_notify script provides a sleek notification system designed specifically for RedM. It features multiple notification types, customizable styling, and optional VORP integration for replacing the default notification system.

✨ Key Features
  • 5 notification types (success, error, warning, info, default)
  • Period-appropriate western styling
  • Optional VORP integration
  • Easy font customization via Google Fonts
  • Customizable images and colors
  • Progress bar with smooth animations
  • Client and server-side exports

Installation

Simple Installation (Standalone)

Use this method if you want to use noc_notify alongside other notification systems. You'll manually call the exports in your scripts.

  1. Drag noc_notify into your resources folder
  2. Add ensure noc_notify to your server.cfg
  3. Restart your server
  4. Use the exports in your scripts (see below)

Advanced Installation (VORP Integration)

Use this method if you want to replace VORP's default notifications with noc_notify. This automatically redirects all VORP notification calls.

  1. Drag noc_notify into your resources folder
  2. Add ensure noc_notify to your server.cfg
  3. Navigate to resources/[VORP]/vorp_core/client
  4. Replace notifications.lua with the one from the VORP Files folder
  5. Navigate to resources/[VORP]/vorp_core/client/exports
  6. Replace exp_notifications.lua with the one from the VORP Files folder
  7. Restart your server
⚠️ Backup First

Before replacing VORP files, make a backup of the original notifications.lua and exp_notifications.lua files in case you need to revert.

Exports

noc_notify uses a simple, unified export system for both client and server sides.

Client-Side Export

exports.noc_notify:Notify(title, message, notifyType, duration)
Parameter Type Required Default Description
title string Yes - Notification title (displayed in Rye font)
message string Yes - Notification message content
notifyType string No 'info' Type of notification (see types below)
duration number No 5000 Display duration in milliseconds

Client Example

-- Success notification
exports.noc_notify:Notify("Success!", "Item purchased successfully", "success", 5000)

-- Error notification
exports.noc_notify:Notify("Error", "Not enough money", "error", 4000)

-- Warning notification
exports.noc_notify:Notify("Warning", "Low health!", "warning", 3000)

-- Default notification
exports.noc_notify:Notify("Shop", "Welcome to the general store", "default")

Server-Side Export

exports.noc_notify:Notify(title, message, notifyType, duration, source)
Parameter Type Required Default Description
title string Yes - Notification title
message string Yes - Notification message content
notifyType string No 'info' Type of notification
duration number No 5000 Display duration in milliseconds
source number Yes - Player server ID to send notification to

Server Example

-- Send notification to a specific player
RegisterNetEvent('myScript:playerDied', function(killerId)
    local victimId = source
    exports.noc_notify:Notify("Death", "You have been killed!", "error", 5000, victimId)
    exports.noc_notify:Notify("Kill", "You killed a player!", "success", 5000, killerId)
end)

Customization

Changing Fonts

One of the easiest ways to customize noc_notify is by changing the fonts. The system uses Google Fonts for easy swapping:

  1. Go to fonts.google.com
  2. Find 2 fonts you like (one for the title, one for the message)
  3. Click "Get Font" on each font card to add to your selection
  4. Click the shopping cart icon in the upper right
  5. Click "Get embed code" and select the @import option
  6. Copy the @import url('...') line
  7. Open nui/notify.css and replace the existing @import at the top
  8. Update the font-family names in .noc-notify-title and .noc-notify-message

Default Fonts

/* Title font - Western style serif */
.noc-notify-title {
    font-family: 'Rye', serif;
}

/* Message font - Clean sans-serif */
.noc-notify-message {
    font-family: 'Barlow', sans-serif;
}

Customizing Images

Replace the notification images in nui/img/:

File Purpose Recommended Size
success.png Success notification icon 96x96
error.png Error notification icon 96x96
warning.png Warning notification icon 96x96
info.png Info notification icon 96x96
default.png Default notification icon 96x96
texture.jpg Background texture overlay 512x512
💡 Image Tips
  • Keep the same filenames when replacing images
  • Use PNG format with transparency for icons
  • Texture images should be seamless for best results

Changing Colors

Modify the colors in nui/notify.css to match your server's theme:

/* Success colors */
.noc-notify.success::before { background: #4E6E45; }
.noc-notify.success .noc-notify-icon { color: #4E6E45; }
.noc-notify.success .noc-notify-progress-bar { background: #4E6E45; }

/* Error colors */
.noc-notify.error::before { background: #8C1C03; }
.noc-notify.error .noc-notify-icon { color: #8C1C03; }
.noc-notify.error .noc-notify-progress-bar { background: #8C1C03; }

/* Warning colors */
.noc-notify.warning::before { background: #EB6B0A; }
.noc-notify.warning .noc-notify-icon { color: #EB6B0A; }
.noc-notify.warning .noc-notify-progress-bar { background: #EB6B0A; }

/* Info colors */
.noc-notify.info::before { background: #FDF1C2; }
.noc-notify.info .noc-notify-icon { color: #FDF1C2; }
.noc-notify.info .noc-notify-progress-bar { background: #FDF1C2; }

/* Default colors */
.noc-notify.default::before { background: #E0A926; }
.noc-notify.default .noc-notify-icon { color: #E0A926; }
.noc-notify.default .noc-notify-progress-bar { background: #E0A926; }

Position & Size

Adjust the notification position and container width in the CSS:

#noc-notify-container {
    position: fixed;
    top: 500px;        /* Vertical position */
    right: 40px;       /* Distance from right edge */
    width: 450px;      /* Container width */
    gap: 15px;         /* Space between notifications */
}
