noc_telegraph

A police activity telegraph board that displays robbery availability based on current police count. Simple, elegant, and period-appropriate.

The noc_telegraph script provides an in-game UI that shows players which robberies are currently available based on the number of police officers on duty. It's designed to work seamlessly with both VORP and Outsider frameworks.

✨ Key Features
  • Real-time police count tracking
  • Configurable robbery requirements
  • Supports VORP and Outsider frameworks
  • Clean, period-appropriate UI design
  • Customizable key binding
  • Lightweight and optimized

Installation

Step 1: Download and Extract

Download noc_telegraph from our store and extract it to your resources folder:

resources/
└── [noc]/
    └── noc_telegraph/

Step 2: Add to server.cfg

Add the following to your server.cfg:

ensure noc_telegraph

Step 3: Configure

Open config.lua to customize the framework setting, key binding, and robbery requirements:

Config = {}

Config.Script = 'out'         -- 'out' = outsider_policeman | 'vorp' = vorp_police
Config.Button = 0x156F7119    -- Backspace (change to your preferred key)

Step 4: Restart Your Server

Restart your server or use the refresh command followed by the resource name:

refresh
ensure noc_telegraph

Configuration

The configuration is straightforward. Open the config.lua file to customize:

Framework Setting

Set the script to work with your chosen framework:

Config.Script = 'out'    -- Options: 'out' (outsider_policeman) or 'vorp' (vorp_police)

Key Binding

Set the key used to open the telegraph board:

Config.Button = 0x156F7119    -- Backspace key hash
💡 Finding Key Hashes

You can find key hashes in the FiveM documentation or by using a key hash finder resource. Common keys include Backspace (0x156F7119), F5 (0x8CC9CD42), etc.

Robbery Configuration

Define which robberies to display and their minimum police requirements:

Config.Robberies = {
    { label = "General Stores",   minPolice = 1 },
    { label = "Secondary Banks",  minPolice = 3 },
    { label = "Full Price Drugs", minPolice = 4 },
    { label = "Jewlery Store",    minPolice = 4 },
    { label = "St. Denis Bank",   minPolice = 5 },
    { label = "Blackwater Bank",  minPolice = 6 },
}

Configuration Parameters

Parameter Type Description
label string The display name shown on the telegraph board
minPolice number Minimum police required for this robbery to be available

Usage

Opening the Telegraph Board

Press and hold the configured key (default: Backspace) to display the telegraph board. The board shows:

  • A list of all configured robberies
  • Whether each robbery is available (green checkmark) or unavailable (red X)
  • Real-time updates based on police count

How It Works

  1. The script monitors police duty status in real-time
  2. When you press the key, it requests current police count from the server
  3. The server calculates availability for each robbery based on minPolice
  4. The UI displays results with visual indicators

Police Count Tracking

The script automatically tracks police officers:

  • Outsider Framework: Checks Player(state).IsPolice and Player(state).IsOnDuty
  • VORP Framework: Checks Player(state).isPoliceDuty

Customization

UI Styling

The telegraph board uses HTML/CSS/JS for its UI. You can customize the appearance by editing:

File Purpose
html/index.html HTML structure of the board
html/style.css Visual styling and colors
html/script.js UI logic and animations

Images

Replace images in html/images/ to customize the look:

  • check.png - Available indicator
  • x.png - Unavailable indicator
  • modern_bg.png - Background texture
  • line_item.png - Row styling
